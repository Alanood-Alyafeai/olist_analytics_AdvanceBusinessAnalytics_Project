# Olist Delivery Prediction Project  
DSAI 4103 — Business Analytics

---

## Overview  
This project analyzes Brazilian e-commerce data to predict late deliveries using machine learning and provide business insights.

---

## Project Structure  
- data/ →  datasets  
- model/ → trained model and scoring script  
- tableau/ → dashboard files and CSVs  
- PowerPoint/ → presentation slides  
- .ipynb → full analysis notebook  

---

## Features  
- Data preprocessing and feature engineering  
- AutoML model (FLAML)  
- SHAP explainability  
- Bias and fairness analysis  
- Tableau dashboard  

---

## Dashboard  
The Tableau dashboard and related files are included in the tableau/ folder (ZIP for submission).

---

## Presentation  
PowerPoint slides are available in the PowerPoint/ folder.
